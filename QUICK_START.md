## 📊 功能演示

### 1. 📈 查看热力图 (每天加入的单词数)

**路径**: `📊 统计分析` → `📈 每日加入热力图` → 点击 `📈 生成热力图`

**输出示例**：
```
📊 每日加入统计热力图
============================================================
🟧 2025-11-02: 3 个单词
🟧 2025-11-03: 3 个单词
...
============================================================
总计: 42 个单词
统计天数: 15 天
平均每天: 2.8 个
```

### 2. ⏰ 查看单词加入时间

**路径**: `📊 统计分析` → `⏰ 单词加入时间记录` → 点击 `⏰ 查看时间记录`

**输出示例**：
```
📝 单词加入时间记录
============================================================
⏰ 2025-11-15 17:34:08: worthless
   📌 无价值的
⏰ 2025-11-15 16:27:08: useless
   📌 无用的
...
```

### 3. 📈 查看最近7天的新增单词

**路径**: `📊 统计分析` → `📈 最近N天的新增单词`

- 输入: `7` (或任何天数)
- 点击 `📈 查看趋势`

**输出**: 最近7天新增的所有单词列表，按时间排序

### 4. 🔤 管理词缀库

#### 添加词缀
**路径**: `🔤 词缀管理` → `➕ 添加词缀`

填写：
- 词缀：`un-` （记得加 `-` 表示前缀）
- 释义：`表示否定或相反`
- 分类：`前缀`
- 点击 `➕ 添加词缀`

#### 查询词缀释义
**路径**: `🔤 词缀管理` → `🔍 查询词缀释义`

输入：`un-`
点击 `🔍 查询`

#### 查看所有词缀
**路径**: `🔤 词缀管理` → `📋 查看所有词缀`

点击 `📋 刷新列表` 查看按分类整理的所有词缀

### 5. 📚 按词缀分类单词

**路径**: `📚 词缀分析` → `📊 按词缀分类单词`

点击 `📊 分析分类`

**输出示例**：
```
【-tion】 (5 个)
  • action: 行动
  • creation: 创造
  • nation: 国家
  • station: 车站
  • prevention: 预防

【-ly】 (4 个)
  • carefully: 小心地
  • happily: 快乐地
  • quickly: 快速地
  • slowly: 缓慢地
...
```

### 6. 🔍 检测潜在词缀

**路径**: `📚 词缀分析` → `🔍 检测潜在词缀`

点击 `🔍 检测`

系统会自动分析词库中的单词，找出高频的词缀模式

**输出示例**：
```
🔍 检测到的潜在词缀
============================================================
• -ion: 出现 5 次
• -tion: 出现 5 次
• -ness: 出现 4 次
• -ly: 出现 4 次
• -ful: 出现 3 次
```

### 7. 💡 Fail跳转建议 (关键功能！)

**路径**: `💡 查询帮助` → `❓ 单词不存在时的建议`

场景：你想查询 "unpredictable" 但词库中没有这个单词

1. 在输入框输入：`unpredictable`
2. 点击 `❓ 获取建议`
3. 系统会建议：

```
❌ 单词 'unpredictable' 不在词库中

💡 您可能想要将这些部分存入词缀库:
============================================================
✗ 'e' - 可添加为新词缀
✗ 'un' - 可添加为新词缀
✗ 'unp' - 可添加为新词缀
✗ 'ble' - 可添加为新词缀

💬 建议: 使用"添加词缀"标签添加这些词缀及其释义
```

4. 根据建议，将合理的词缀（如 "un-" 和 "-able"）添加到词缀库
5. 下次遇到相同词缀的单词时，系统会自动识别和分类

## 🧠 学习场景示例

### 场景 A: 从头构建个人词缀库

1. **第一天**: 添加10个随机单词
   ```
   unbelievable, unable, unforgettable, ...
   ```

2. **第二天**: 打开 `📚 词缀分析` 检测潜在词缀
   - 系统建议: `un-`, `-able`, `-tion` 等

3. **第三天**: 逐个添加推荐的词缀
   - 输入词缀定义
   - 记录释义

4. **第四天**: 查看分类
   - 使用 `📊 按词缀分类单词` 看单词的分布
   - 发现模式和规律

5. **持续学习**: 每天查看热力图，跟踪学习进度

### 场景 B: 快速处理未知单词（ACAM流程）

1. **遇到新单词**: "disrespectful"
2. **在精确查询中查询**: 未找到
3. **打开 Fail 跳转建议**
4. **系统建议**: 可能的词缀部分
5. **你识别出**: `dis-` (否定), `-ful` (充满)
6. **添加新词缀**: 使用 `➕ 添加词缀` 添加 `dis-`
7. **完整体验**: 现在系统知道了新词缀，下次遇到同样词缀的单词会自动分类

## 💡 专业用法提示

### 📌 词缀命名规范

- **前缀**: 用 `-` 作后缀标记，如 `un-`, `re-`, `dis-`
- **后缀**: 用 `-` 作前缀标记，如 `-tion`, `-ly`, `-ness`
- **中缀**: 用 `-` 前后标记，如 `-ing-` （极少见）

### 📌 时间查询格式

- 日期格式：`YYYY-MM-DD`（如 `2025-11-14`）
- 可只指定开始或结束日期
- 范围查询示例：
  - 开始：`2025-11-01`，结束：`2025-11-15` → 查询这15天的单词
  - 开始：`2025-11-10`，结束：（空）→ 查询11月10日之后的单词

### 📌 热力图解读

- 🟩 绿色：1 个单词（低频）
- 🟧 橙色：中等频率
- 🟥 红色：高频率（最多的单词）
- ⬜ 白色：零单词（某天没有添加）

## 🔧 常见问题

**Q: 为什么词缀分类中有 "[无匹配词缀]"？**
A: 这些单词的前缀或后缀还没有在词缀库中定义。使用 Fail 建议功能或 🔍 检测功能来发现它们。

**Q: 如何备份我的词库和词缀库？**
A: 
- 词库：`vocabulary.json` 
- 词缀库：`affixes.json`
- 直接复制这两个文件即可

**Q: 能否导出热力图？**
A: 暂时需要截图。你也可以在 `📊 统计分析` 中复制文本格式的热力图。

**Q: 时间戳的时区是多少？**
A: 使用你系统的本地时区（ISO 8601 格式）。
